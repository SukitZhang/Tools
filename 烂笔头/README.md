.tar.gz格式（.tar是打包! .gz是压缩！）
解压：[＊＊＊＊＊＊＊]$ tar zxvf FileName.tar.gz
压缩：[＊＊＊＊＊＊＊]$ tar zcvf FileName.tar.gz DirName


远程拷贝到本地：scp root@vps.jandou.com:../home/wwwroot/ss.jandou.com.tar.gz ~/Desktop

本地上传到远程：scp ~/ root@vps.jandou.com:../home

>  ~/ 当前用户目录


删除空目录dir: `rmdir dir`

删除非空目录dir: `rm -rf dir`
